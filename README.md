<h1 align="center">🎮 App de minijuegos mobile</h1>

<h3>Aplicación mobile desarrollada en Flutter que combina un sistema de logros con minijuegos, enfocada en la interacción del usuario y progresión dentro de la app. </h3>

> <h2>🎯 Objetivo</h2>

Explorar el desarrollo mobile con Flutter implementando mecánicas de juego y un sistema de recompensas basado en acciones del usuario. El proyecto está enfocado en la lógica de estados, interacción del usuario y manejo de progresión dentro de una aplicación mobile.

> <h2>🚀 Funcionalidades</h2>
<ul>
  <li>Sistema de login simple</li>
  <li>Minijuegos integrados</li>
  <li>Sistema de logros y recompensas</li>
  <li>Progresión del usuario</li>
  <li>Persistencia de datos</li>
</ul>
